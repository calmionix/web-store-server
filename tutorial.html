Cara Membeli Rank & Shard

Panduan lengkap untuk pembelian di Calmionix

Langkah-langkah Pembelian

1

Pilih produk (Rank / Shard)

Kunjungi halaman toko dan pilih produk yang ingin kamu beli. Kami menyediakan berbagai rank dan paket shard.

2

Klik "Beli Sekarang"

Setelah memilih produk, klik tombol "Beli Sekarang" pada kartu produk.

3

Masukkan username Minecraft

Isi username Minecraft kamu dengan benar. Pastikan tidak ada typo karena reward akan dikirim ke username tersebut.

4

Lanjutkan ke pembayaran

Klik "Lanjut Pembayaran" dan ikuti instruksi pembayaran yang tersedia (QRIS, Transfer Bank, E-Wallet).

5

Setelah pembayaran berhasil, reward otomatis masuk

Setelah pembayaran dikonfirmasi, rank atau shard akan otomatis masuk ke akun Minecraft kamu dalam beberapa menit.

Metode Pembayaran

Kami mendukung berbagai metode pembayaran:

📱 QRIS

Scan QR code dengan aplikasi e-wallet atau mobile banking kamu.

🏦 Transfer Bank

Transfer ke rekening bank yang tersedia (BCA, BNI, BRI, Mandiri).

💳 E-Wallet

Gopay, OVO, DANA, LinkAja, dan ShopeePay.

Jika mengalami masalah, silakan tanya di Discord

Tim support kami siap membantu 24/7

Panduan Integrasi Payment

Bagian ini berisi panduan teknis untuk mengintegrasikan sistem pembayaran.

1. Lokasi Integrasi

Semua logic pembayaran ada di file checkout.js pada fungsi processPayment().

// File: js/checkout.js

function processPayment(productId, username) {
  const product = products[productId];

  // ========================================
  // PAYMENT INTEGRATION AREA
  // ========================================
  // 1. Connect to Midtrans / Tebex
  // 2. Send transaction data to backend
  // 3. Handle success response
  // 4. Trigger Minecraft reward (rank/shard)
}

2. Metode Tebex

Tebex adalah solusi all-in-one untuk Minecraft server store. Plugin otomatis memberikan reward setelah pembayaran.

Langkah-langkah:

  1. Buat akun di tebex.io
  2. Hubungkan dengan server Minecraft kamu
  3. Install plugin Tebex di server
  4. Buat package untuk setiap rank/shard
  5. Set command yang akan dijalankan (contoh: lp user {username} parent add vip)
  6. Ganti link "Beli Sekarang" dengan link checkout Tebex
// Contoh: Redirect ke Tebex checkout
function processPayment(productId, username) {
  const tebexUrl = `https://checkout.tebex.io/your-store/${productId}?username=${username}`;
  window.location.href = tebexUrl;
}

3. Metode Midtrans

Midtrans adalah payment gateway lokal Indonesia. Memerlukan backend untuk handle webhook.

Langkah-langkah:

  1. Buat akun di midtrans.com
  2. Dapatkan Server Key dan Client Key
  3. Buat backend endpoint untuk create transaction
  4. Integrasi Snap.js di frontend
  5. Setup webhook untuk handle payment notification
  6. Kirim command ke Minecraft server setelah pembayaran sukses
// 1. Create transaction di backend
const response = await fetch('/api/create-transaction', {
  method: 'POST',
  body: JSON.stringify({ productId, username })
});
const { token } = await response.json();

// 2. Open Midtrans Snap
window.snap.pay(token, {
  onSuccess: function(result) {
    // Trigger Minecraft reward
  }
});

4. Integrasi Minecraft

Gunakan LuckPerms untuk memberikan rank. Gunakan RCON atau plugin untuk eksekusi command.

Opsi 1: LuckPerms + Webhook

// Setelah pembayaran sukses, kirim command:
`lp user ${username} parent add ${rank}`

// Bisa menggunakan:
// - RCON langsung ke server
// - Plugin API endpoint
// - Tebex plugin (otomatis)

Opsi 2: RCON Node.js

const Rcon = require('rcon');

const rcon = new Rcon('localhost', 25575, 'password');

await rcon.connect();
await rcon.send(`lp user ${username} parent add ${rank}`);
await rcon.disconnect();
